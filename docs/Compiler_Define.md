# Compiler_Define.h

## 宏定义
```cpp
声明：
CJF2_NBT_CPP_COMPILER_MSVC
说明：
为1则代表当前编译器是msvc
为0则代表当前编译器不是msvc


声明：
CJF2_NBT_CPP_COMPILER_GCC
说明：
为1则代表当前编译器是gcc
为0则代表当前编译器不是gcc


声明：
CJF2_NBT_CPP_COMPILER_CLANG
说明：
为1则代表当前编译器是clang
为0则代表当前编译器不是clang


声明：
CJF2_NBT_CPP_COMPILER_NAME
说明：
字符串值，代表编译器名称
未知则为"Unknown"
```
